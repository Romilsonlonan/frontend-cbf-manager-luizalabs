# **App Name**: Backend Buddy

## Core Features:

- Firebase Studio Access: Direct access to Firebase Studio via provided URL (https://studio.firebase.google.com/backend-95170514).
- Credential Management: Securely store and manage Firebase credentials.
- Firebase Service Dashboard: UI to monitor and manage various Firebase services (e.g., Authentication, Firestore, Storage).

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a trustworthy, reliable feel. This choice aligns with the cloud computing aspects of Firebase, often associated with dependable infrastructure.
- Background color: Light gray (#F5F5F5) for a clean, neutral workspace.
- Accent color: Teal (#009688) to highlight interactive elements and important data, complementing the primary blue and enhancing the user experience.
- Body and headline font: 'Inter', a sans-serif, provides a modern, clean and easily readable interface.
- Use clear and recognizable icons to represent Firebase services and actions.
- Employ a modular layout with distinct sections for different Firebase services, ensuring easy navigation and focus.
- Subtle transitions and animations to provide feedback and enhance user interaction.
